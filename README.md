# TypeRush - Typing Speed Test

## Description

TypeRush is a responsive web application designed to test and improve users' typing speed and accuracy. The application displays random texts based on selected difficulty levels and provides real-time feedback while typing.

The project was developed using only HTML5, CSS3, and Vanilla JavaScript without any external libraries or frameworks.

## Live Demo

🔗 https://bespoke-lily-ff675a.netlify.app/

---

## Features

- Random text generation from a library of texts.
- Three difficulty levels:
  - Easy
  - Medium
  - Hard
- Real-time character highlighting:
  - Correct characters are displayed in green.
  - Incorrect characters are displayed in red.
- Automatic timer that starts on the first keystroke.
- Words Per Minute (WPM) calculation.
- Typing accuracy percentage calculation.
- Result panel displayed after completing the test.
- Performance feedback message based on typing speed.
- "New Test" button to restart the test without refreshing the page.
- Best score saving using LocalStorage.
- Fully responsive design for desktop and mobile devices.
